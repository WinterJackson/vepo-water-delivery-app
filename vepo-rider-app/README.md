# Vepo Rider App

## Required Environment Variables

To run the app securely without committing secrets to the repository, create a `.env` file in the root of the `vepo-rider-app` folder. 

Refer to the structure below for the required keys. Do **not** commit the values of these variables.

```env
# ── Required ──────────────────────────────────────────────
EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=""
EXPO_PUBLIC_BACKEND_BASE_URL=""

# ── Optional / Third Party ────────────────────────────────
EXPO_PUBLIC_GOOGLE_MAPS_KEY=""
```
